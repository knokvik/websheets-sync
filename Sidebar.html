API Key Trouble

Your API key may not save because this browser is signed in to more than one Google account.

Try one of these fixes:

  1. Sign out of extra Google accounts.
  2. Use an incognito window with only this Google account signed in.
  3. Make sure your default Google account is the same one that owns this sheet.

Guide

Exa Agent

Use this when you want Exa to create a full table.

Open the Exa Agent tab, write what table you want, then click Run Agent. The table starts at your selected cell unless you choose another cell in More options.

Example: Find top 40 AI companies with website, CEO, founding date, and headquarters.

EXA formula

Use this inside a cell when you want one answer for one row.

Basic format: =EXA("what you want", cell)

=EXA("Return only the company website URL", A2)
=EXA("Return only the CEO name", A2)

The second part, like A2, is the cell Exa should use as context. You can fill the formula down a column to run it for many rows.

Batch

Select cells with Exa formulas, then use Batch to refresh them or convert them to values.

Convert to values when you want to keep the current results and stop the formulas from running again.
